# Payments Page

This page was made using React 18, Material UI and Tailwind. This page is fully _responsive_.

## Instructions
1. install dependencies using ```npm install```
2. use ```npm start``` to start the project

## Dependencies Used
1. Tailwind
2. Material UI
3. gh-pages
